# Local Data Folder

This folder is intentionally kept empty in version control.

Place local video files under `data/` (subfolders supported), for example:

```
data/
  Recordings/
    subject_01/
      clip_001.mp4
```

Supported video extensions:

`.mp4`, `.mov`, `.avi`, `.mkv`, `.m4v`

Do not commit participant recordings or any personally identifiable media.
